# Lab 6 – PHP Function Challenges

A PHP exercise practicing arrow functions, array iteration, and string manipulation through three short coding challenges.

## Challenges

1. **Fahrenheit to Celsius** – Converts a temperature using an arrow function.
2. **Print Names in Uppercase** – Loops through an array and prints each name in uppercase.
3. **Find the Longest Word** – Splits a sentence and returns the longest word.

## Usage

Run `Lab6.php` on any PHP-enabled server or local environment.
