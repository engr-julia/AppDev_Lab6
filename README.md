# Lab 6 – PHP Function Challenges

A PHP exercise practicing arrow functions, array iteration, and string manipulation through three short coding challenges. This lab explores different ways to define and use functions in PHP, from concise arrow functions to traditional named functions.

## Challenges

1. **Fahrenheit to Celsius** – Converts a temperature value using a PHP arrow function (`fn`). Demonstrates how arrow functions offer a clean, concise syntax for simple expressions.

2. **Print Names in Uppercase** – Accepts an array of names and loops through each one, printing it in uppercase using `strtoupper()`. Demonstrates passing arrays as function arguments and using `foreach`.

3. **Find the Longest Word** – Takes a sentence string, splits it into words using `explode()`, and iterates through them to find and return the longest word. Demonstrates string manipulation and comparison logic.

## Concepts Covered

- Arrow functions (`fn`) vs. named functions
- Array iteration with `foreach`
- String functions: `strtoupper()`, `explode()`, `strlen()`
- Returning values from functions

## Usage

Run `Lab6.php` on any PHP-enabled server or local environment such as XAMPP, MAMP, or the built-in PHP development server.

```bash
php Lab6.php
```
